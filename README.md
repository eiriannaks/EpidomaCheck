💵 EpidomaCheck - The easy way to check!
Το EpidomaCheck είναι μια προηγμένη Web εφαρμογή (Fintech-Grade Web App) που προσφέρει στους Έλληνες πολίτες έναν ταχύτατο και έγκυρο τρόπο να εντοπίζουν τα επιδόματα που δικαιούνται. Με το νέο Altgage Design, η εφαρμογή προσφέρει μια επαγγελματική και φιλική εμπειρία χρήστη.
🌟 Τι νέο υπάρχει στην έκδοση v83.0
⚡ Altgage Progressive Flow: Η εφαρμογή ακολουθεί μια σταδιακή δομή πλοήγησης (Landing -> Form -> Results), μειώνοντας τον φόρτο πληροφορίας και καθοδηγώντας τον χρήστη βήμα-βήμα.
🤖 Babis Executive AI: Ο ψηφιακός βοηθός «Μπάμπης» είναι πλέον πιο έξυπνος. Διαθέτει ανασχεδιασμένο Blue Vision UI και μπορεί να ερευνήσει νέα επιδόματα με μια απλή αναφορά του ονόματός τους.
☁️ Cloud Intelligence & Sync: Χρήση της βάσης δεδομένων Firebase για αυτόματη αποθήκευση και συγχρονισμό νέων επιδομάτων που αναγγέλλουν οι χρήστες.
🎨 Emerald Inversion Design: Μοντέρνα αισθητική με Emerald Green φόντο, Plus Jakarta Sans τυπογραφία και απόλυτα αιχμηρές (sharp) γωνίες για Fintech αίσθηση.
📍 Διευρυμένα Κριτήρια: Υποστήριξη για ειδικές κατηγορίες όπως Μονογονείς, Μειονότητες, Τρίτεκνοι/Πολύτεκνοι και Αγρότες.
🛠️ Βασικές Λειτουργίες
🔍 Έξυπνος Έλεγχος: Αυστηρά φίλτρα βάσει προφίλ (Φύλο, Ηλικία, Εισόδημα, Περιουσία).
🤰 Μητρότητα & Προστασία: Ειδική μέριμνα για επιδόματα μητέρων με αυστηρό ηλικιακό όριο (έως 52 ετών).
📢 Αναγγελία Νέου Επιδόματος: Οι χρήστες μπορούν να «προτείνουν» ένα επίδομα και ο Μπάμπης AI αναλαμβάνει την έρευνα και την αυτόματη καταχώρηση στη βάση.
📜 Πλήρης Κατάλογος: Άμεση πρόσβαση σε 35+ επίσημες παροχές (Α21, Στέγαση, ΕΕΕ, Youth Pass κ.α.).
🔝 Smart Navigation: Κουμπί επιστροφής στην κορυφή (Scroll to top) και δυναμικά badges προθεσμιών (Ανοιχτό/Κλείνει σύντομα/Εληξε).
🚀 Τεχνολογίες
Frontend: HTML5 & Tailwind CSS (Custom Altgage Components).
Intelligence: Gemini AI API (gemini-2.5-flash-preview) για τη λειτουργία του Μπάμπη.
Database: Firebase Firestore για real-time συγχρονισμό δεδομένων.
Auth: Firebase Anonymous Authentication για ασφαλή πρόσβαση στη βάση.
📦 Εγκατάσταση & Χρήση
Το project είναι Single-File Application (SFA), που σημαίνει ότι όλος ο κώδικας βρίσκεται στο index.html.
Κατεβάστε το αρχείο index.html.
Ανοίξτε το σε οποιονδήποτε σύγχρονο browser.
Για τη λειτουργία του Cloud Sync, η εφαρμογή απαιτεί σύνδεση στο διαδίκτυο.
