💵 EpidomaCheck - Ο Ψηφιακός Βοηθός Επιδομάτων

Το EpidomaCheck είναι μια σύγχρονη web εφαρμογή που βοηθά τους Έλληνες πολίτες να ανακαλύψουν άμεσα ποια επιδόματα δικαιούνται από το gov.gr, την ΗΔΙΚΑ και τον ΟΠΕΚΑ.

Με τη βοήθεια της Riri AI, η διαδικασία αναζήτησης γίνεται απλή, γρήγορη και απόλυτα προσωποποιημένη.

🌟 Βασικές Λειτουργίες

🐱 Riri AI Assistant: Ένας ενσωματωμένος ψηφιακός βοηθός που απαντά σε ερωτήσεις και αναγνωρίζει αυτόματα links από το gov.gr.

🔍 Έξυπνα Φίλτρα: Αυστηρό φιλτράρισμα βάσει φύλου, ηλικίας, εισοδήματος και οικογενειακής κατάστασης.

👨‍👩‍👧‍👦 Υπολογισμός Νοικοκυριού: Δυναμική προσαρμογή των εισοδηματικών ορίων (π.χ. για ΕΕΕ, Επίδομα Στέγασης) βάσει των μελών της οικογένειας.

🌓 Dark Mode: Ξεκούραστη πλοήγηση με εναλλαγή μεταξύ φωτεινού και σκοτεινού θέματος.

📄 PDF & Εκτύπωση: Δυνατότητα αποθήκευσης των αποτελεσμάτων για μελλοντική χρήση.

📢 Αναγγελία Επιδομάτων: Μόνιμη φόρμα για αυτόματη ανάλυση νέων επιδομάτων μέσω AI.

🛠️ Τεχνολογίες

Frontend: HTML5, Tailwind CSS (για μοντέρνο και responsive σχεδιασμό).

Intelligence: Gemini AI API (μοντέλο flash-preview) για τη λειτουργία της Riri.

Icons: Font Awesome & Emoji.
